intro

golor is a command line tool for golang source code coloring

how to build

git clone 'git://github.com/hantuo/golor.git'

cd golor

./build.sh

usage

cat src/hantuo.org/golor/golor.go | bin/golor

or

bin/golor src/hantuo.org/golor/golor.go
